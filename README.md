# RRRRRRRRespawn

[Bevy Game Jam #4](https://itch.io/jam/bevy-jam-4)

![](assets/atlas/title.png)

**Instructions**

- Collect all stars

**Controls**

- Left / A - Move Left
- Right / D - Move Right
- Space - Jump
- R - Reset
- X - Split into two
